# MapReduce Project – Movie Genre Count

## Project Description

This project demonstrates the basic working of the MapReduce programming model using Python.

The project reads movie genre data from an input file and calculates the number of occurrences of each genre.

The MapReduce process consists of three main stages:

- Mapper
- Partitioner
- Reducer

The `main.py` file controls the complete execution of the project.
